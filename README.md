## Maven Repository

The `wwmm-parent` POM is also available via Maven, and is on [Maven Central](https://search.maven.org/):

```xml
<dependency>
  <groupId>uk.ac.cam.ch.wwmm</groupId>
  <artifactId>wwmm-parent</artifactId>
  <version>6</version>
</dependency>
```
